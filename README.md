Projet 1: Gestion de notes des étudiants

Projet realise par A SIMO NGOUNOU FRANK
              Matricule : 2751695

FONCTIONNALITE DE L APPLICATION 

- CREATION D UN ETUDIANT, L UTILISATEUR DOIT SAISIR LE MATRICULE, LE NOM ET LE PRENOM DE L ETUDIANT
- STOCKAGE DES ETUDIANTS DANS UN FICHIER CSV APPELE REPERTOIRE_ETUDIANTS
- SAISI DES NOTES D UN ETUDIANT DEJA ENREGISTRE, L UTILISATEUR DOIT SAISIR TOUTES LES NOTES DE L ETUDIANT 
- CLASSEMENT DES ETUDIANTS PAR ORDRE DE MERITE DANS UN FICHIER PERSISTANT CSV APPELE RESULTATS_ETUDIANTS
